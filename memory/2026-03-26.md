# 2026-03-26 — Likely You Product Deep Dive

## Product vs Competitors

### Co-Star / The Pattern = emotional companion tools
- Push one vague emotional guide daily, like a horoscope weather report
- Don't need to be accurate — vagueness IS the feature (Barnum effect)
- Core experience is "today", no timeline
- Users want daily emotional companionship ("what should I do today")
- High frequency low stickiness, open and glance daily
- Social currency driven ("I'm a Scorpio, what are you")
- Don't care about accuracy, care about feeling understood

### Likely You = trust machine
- Trade past accuracy for trust, monetize future curiosity
- MUST be accurate — users can verify each year themselves
- Core experience is TIMELINE: past 5 years + current year + future + major cycles (大运)
- Users want to understand a person (is this date reliable, what's up with my boss)
- Event-driven: use it when meeting someone new, not daily
- Trust-driven: "past was accurate, what about future?" -> pay
- Accuracy matters because it's the prerequisite for payment

### One-liner
Co-Star sells daily companionship. We sell verifiable precision.
Their users ask "how's today". Our users ask "what's this person about, when does my luck turn".

## Product Architecture
- 518,400 personality types (not 12 zodiac, not 16 MBTI)
- Three-card system: personality card (sharing/virality) -> annual card (validation) -> K-line card (paid)
- Tech: Python calculation engine + Next.js + Supabase + Flask API + Vercel/Railway
- 97K lines of code, solo developer, 5 months from zero Python knowledge
- MMA vertical as zero-competition entry point
